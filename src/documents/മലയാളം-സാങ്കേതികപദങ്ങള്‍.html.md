താഴെയുള്ള പദങ്ങള്‍ എന്റെ സ്വന്തം നിഘണ്ടുവില്‍ ചേര്‍ക്കപ്പെട്ടിരിക്കുന്നു.

computer = സകലകലായന്ത്രം

inernet = യന്ത്രംഖല

web = ഖല

website = ഖലയിടം

html (hypertext markup language) = അടയാളിത പുഷ്ട ഖലഭാഷ (പുഷ്ട ഭാഷ എന്നു് ചുരുക്കാവുന്നതാണു്)

code = യന്ദ്രേശം
